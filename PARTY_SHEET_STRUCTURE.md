# 파티 정보 관리 시트 구조

## 시트 이름: "파티 일정"

### 컬럼 구조 (헤더)

**중요**: 첫 번째 행에 아래 헤더를 정확히 입력하세요.

| 컬럼명 | 설명 | 예시 | 필수 | 비고 |
|--------|------|------|------|------|
| 날짜 | YYYY-MM-DD 형식 | 2025-12-19 | ✅ | 반드시 이 형식으로 입력 |
| 년 | 연도 | 2025 | | 자동 계산 가능 (선택) |
| 월 | 월 (1-12) | 12 | | 자동 계산 가능 (선택) |
| 일 | 일 (1-31) | 19 | | 자동 계산 가능 (선택) |
| 파티타입 | sulgaeting, hexagon-party, star-party | sulgaeting | ✅ | 정확히 입력 |
| 상태 | 모집중, 마감 | 모집중 | ✅ | 정확히 입력 |
| 제목 | 파티 제목 | 술개팅 | | |
| 시간 | 시간 정보 | 19:30 ~ 21:30 | | |
| 나이범위 | 나이 범위 | 89년생 ~04년생 | | |
| 남성인원 | 남성 인원 수 | 7 | | 숫자만 입력 |
| 여성인원 | 여성 인원 수 | 10 | | 숫자만 입력 |
| 비고 | 추가 정보 | 여자 마감 | | |

### 사용 방법

1. **Google Sheets에서 "파티 일정" 시트 생성**
   - 같은 스프레드시트에 새 시트 추가
   - 시트 이름을 정확히 "파티 일정"으로 설정

2. **헤더 입력**
   - 첫 번째 행에 위의 컬럼명을 정확히 입력
   - 순서는 중요하지 않지만, 컬럼명은 정확히 일치해야 함

3. **데이터 입력**
   - 각 파티 정보를 행 단위로 추가
   - 날짜는 반드시 `YYYY-MM-DD` 형식 (예: `2025-12-19`)

4. **자동 반영**
   - 웹사이트를 새로고침하면 자동으로 반영됨
   - 코드 수정 없이 시트에서만 관리 가능

### 주의사항

- **날짜 형식**: 반드시 `YYYY-MM-DD` 형식으로 입력 (예: `2025-12-19`)
- **파티타입**: 정확히 입력해야 함
  - `sulgaeting` (소문자)
  - `hexagon-party` (하이픈 포함)
  - `star-party` (하이픈 포함)
- **상태**: `모집중` 또는 `마감`만 입력 (대소문자 구분)
- **빈 셀**: 빈 셀은 자동으로 처리되므로 선택 사항은 비워둬도 됨
- **년/월/일 컬럼**: 날짜에서 자동 계산 가능하지만, 수동 입력도 가능

### 시트 생성 예시

```

A2: 2025-12-19  B2: 2025  C2: 12    D2: 19    E2: sulgaeting    F2: 모집중   G2: 술개팅   ...
A3: 2025-12-20  B3: 2025  C3: 12    D3: 20    E3: hexagon-party F3: 모집중   G3: 육각형 파티 2기 ...
```
